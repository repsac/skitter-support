# Skitter Support

**Skitter** is a guitar and bass finger coordination practice app for iPhone and iPad. It provides visual guidance synchronized to a metronome, tracks your progress over time, and adapts to your skill level with structured progressions, session templates, and a guided first-week program.

## About

Skitter is an independent app by Ed Caspersen. The metronome engine is built on [AudioKit](https://audiokit.io/) with a custom rhythm grading pipeline. The fretboard renderer is a fully custom SwiftUI view supporting 4- and 6-string instruments, left-handed mode, and three dot-label modes.
