# User Guide

A walkthrough of everything Skitter can do. For quick fixes to common issues, see [Troubleshooting](troubleshooting.md).

## Getting Started

When you first open Skitter, the Guided First Week program walks you through a short daily exercise plan designed for beginners. You can skip it and jump straight to free practice if you prefer.

The **Practice** tab is the main screen. From here you can:

- Start a quick exercise from the suggestions or exercise library
- Launch a saved preset, template, or progression
- Continue the Guided First Week program

## Exercises

Skitter includes five categories of finger coordination exercises:

- **Spiders**: sequential finger patterns that build independence across all four fretting fingers
- **Scales**: major, minor, pentatonic, blues, and other common scale shapes
- **Arpeggios**: chord-tone patterns played one note at a time
- **Intervals**: two-note patterns that train your ear and hand simultaneously
- **Chord Shapes**: static chord forms shown on the fretboard for visual memorization

Each exercise has a default BPM, duration, and starting fret that you can adjust before starting.

## Display Modes

Skitter offers three ways to see what to play:

- **Fretboard**: a visual fretboard diagram with colored dots showing where to place your fingers. The current note is highlighted; lead-in notes preview what's coming.
- **Tab**: standard guitar tablature notation scrolling in time with the metronome.
- **Notation**: standard music notation for those who read sheet music.

Switch between modes in **Settings → Display → Default View**, or change on the fly during session setup.

## The Metronome

Every exercise is synchronized to a metronome. The default BPM is set in **Settings → Metronome → Default BPM** and can be adjusted per session.

Skitter includes several built-in click sounds. You can also import your own custom sounds from audio files on your device.

## Rhythm Grader

The rhythm grader listens to your playing through the microphone and scores how accurately you play each note on the beat.

- **Timing Sensitivity**: controls how strict the grading window is (Relaxed, Standard, or Strict). Set in **Settings → Rhythm Grader → Timing Sensitivity**.
- **Note Accuracy**: optionally validates that you played the correct pitch, not just the correct timing. Enable in **Settings → Rhythm Grader → Grade Note Accuracy**.
- **Feedback Style**: choose how accuracy is shown on the fretboard during practice — colored shapes, a simple checkmark trail, or nothing.

The grader requires microphone permission. If denied, exercises still work normally without grading.

## Progressions

A progression is a structured practice plan that automatically increases difficulty over time.

- Choose one or more exercise types
- Set a starting and target BPM
- Set a starting and target duration
- Choose a ramp style (proportional or linear)
- Optionally require a minimum accuracy score before advancing

Progressions track your position so you pick up where you left off each session.

## Session Templates

A template is a custom sequence of exercises that play back-to-back, like a playlist for your practice session.

- Add exercises in any order with per-exercise BPM and duration
- Set rest bars between exercises
- Enable auto-advance to move to the next exercise automatically
- Run the entire template as a single session

## Presets

Save your favorite exercise configuration (exercise type, BPM, fret, duration mode, grading settings) as a preset for one-tap launching from the Practice screen.

## Trophies

Skitter awards trophies for practice milestones — your first session, practice streaks, BPM records, total hours logged, and more. View your collection on the **Trophies** tab.

## Settings

All preferences are in the **Settings** tab:

- **Instrument**: switch between guitar and bass, select a tuning preset, or create custom tunings
- **Practice Defaults**: starting fret, count-in beats, default reps, direct-to-session mode
- **Display**: appearance, default view, dot labels, left-handed mode, lead-in guide, fretboard motion
- **Metronome**: default BPM, click sound
- **Rhythm Grader**: enable by default, timing sensitivity, feedback style
- **Tuner**: reference pitch (A4)
- **Audio Input**: choose which microphone the grader and tuner use
- **Encouragement**: toggle welcome messages on the Practice screen
- **Notifications**: Key-of-the-Day daily practice suggestion
- **Goals**: weekly practice minute target
- **Data & Privacy**: iCloud sync, Dropbox backup, storage info, clear history

## Data & Sync

Skitter stores all practice data locally on your device.

- **iCloud Sync**: syncs sessions, progressions, templates, presets, trophies, and settings across all your Apple devices signed into the same iCloud account. Toggle in **Settings → Data & Privacy → iCloud Sync**. Changes take effect on next app launch.
- **Dropbox Backup**: backs up your data to a private Dropbox App folder for cross-platform portability. Connect in **Settings → Data & Privacy → Dropbox Backup → Connect**. Syncs automatically when the app opens, backgrounds, or after completing an exercise.
